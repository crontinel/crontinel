## Ready to Apply

### 2. 1Password for Open Source
- **URL:** https://github.com/1Password/for-open-source
- **What you get:** Free 1Password Teams account (does not expire as long as project is eligible)
- **Status:** Not applied
- **Eligibility:** Must be a core contributor to an OSS project that is 30+ days old, active, permissive license. Crontinel qualifies.
- **How to apply:**
  1. Start a free 14-day trial of 1Password Teams
  2. Invite at least one other person to the Owners group
  3. Fill out the application form and open an issue on the 1Password/for-open-source repo
- **Notes:** Great for credential management. Free forever for qualifying OSS projects.

### 3. Notion for Startups
- **URL:** https://www.notion.com/startups
- **What you get:** Up to $500-$6,000 in credits (6 months free on Business plan with partner affiliation)
- **Status:** Not applied
- **Eligibility:** Non-paying Notion customer, valid work email (not Gmail), fewer than 50 employees, under $10M funding. Crontinel qualifies.
- **How to apply:** Fill out the form at notion.com/startups. Use info@crontinel.com. Response in 3-5 business days.
- **Notes:** Useful for internal docs and project management. At minimum, $500 credit available without partner affiliation.

### 4. Zoho Startup Program
- **URL:** https://www.zoho.com/biz/
- **What you get:** Free or discounted Zoho suite (already using Zoho Mail)
- **Status:** Not applied
- **Notes:** Already using Zoho Mail for info@crontinel.com MX records. Explore Zoho CRM (for tracking leads), Zoho Analytics (for usage data), or Zoho Projects. Low priority since most needs are covered by other tools.

## Already Available (No Application Needed)

### 5. GitHub Actions (Free for Public Repos)
- **URL:** https://github.com/pricing
- **What you get:** Unlimited GitHub Actions minutes for public repositories
- **Status:** Already available
- **Notes:** crontinel/crontinel is a public repo, so CI/CD is free and unlimited. Private repos on the Free plan get 2,000 minutes/month. As of March 2026, there is a $0.002/min cloud platform charge for self-hosted runners, but standard runners on public repos remain free.

### 6. GitHub Enterprise (Open Source)
- **URL:** https://github.com/enterprise
- **What you get:** GitHub doesn't offer free Enterprise for OSS specifically, but public repos already get most features
- **Status:** Not needed
- **Notes:** Public repos already get unlimited Actions, Packages, and Pages. GitHub Advanced Security (code scanning, secret scanning) is also free for public repos. No need to apply for Enterprise.
